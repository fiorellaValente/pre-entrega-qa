# Automatización QA - Pre-Entrega

## Descripción
Suite de tests automatizados para validar la funcionalidad de login en SauceDemo usando Selenium y Pytest.

## Tecnologías
- **Selenium WebDriver 4.15**
- **Pytest 7.4**
- **Python 3.8+**
- **WebDriver Manager**

## Instalación

```bash
# Instalar dependencias
pip install -r requirements.txt
```
